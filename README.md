`SurrogateBMA` is an `R` package that estimates the proportion of treatment effect explained by a surrogate marker using a Bayesian Model Averaging (BMA) approach. 

More details on the method are available in Duan, Y., & Parast, L. (2024). Flexible evaluation of surrogate markers with Bayesian model averaging. Statistics in medicine, 43(4), 774-792. [doi:10.1002/sim.9986](https://doi.org/10.1002/sim.9986). 

Go here to view a tutorial for this package: [SurrogateRank Tutorial](https://htmlpreview.github.io/?https://github.com/laylaparast/SurrogateBMA/blob/main/SurrogateBMA_tutorial.html).
